# Daily Affirmation Dashboard

**Daily Affirmation Dashboard** â€” a minimalist, mobile-first, offline-ready web app (PWA) that shows daily affirmations, allows adding/saving your own, and can be installed on Android.

## Features
- New affirmation on load and "Next" / "Surprise" controls.
- Add and save personal affirmations to `localStorage`.
- Dynamic background that changes by time of day.
- Live date & time widget.
- Optional weather display (uses OpenWeatherMap; requires API key).
- Works fully offline via Service Worker.
- PWA manifest for "Add to Home Screen".
- Designed for GitHub Pages hosting and later wrapping for Android via Bubblewrap/Capacitor.
