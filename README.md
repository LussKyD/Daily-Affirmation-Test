# Daily Affirmation Dashboard

Minimal, sleek, mobile-first Daily Affirmation PWA.  
Designed to be beautiful, offline-capable, and "AI-driven" on-device.

## Features
- On-device AI-like generator that composes uplifting affirmations.
- Add your own affirmations and save to `localStorage`.
- Dynamic, sleek iPhone-like UI.
- PWA-ready: manifest + service worker.
- Works offline once cached via service worker.
- Easy to host on GitHub Pages.

## Quick start
1. Clone or download this repo.
2. Serve locally (https or localhost required for service worker):
   - `python -m http.server 8000`
3. Open `http://localhost:8000` and test the app.
4. Push to GitHub and enable Pages (root) for deployment.

## Icons
Placeholder icons are included in `/icons`. Replace them with branded PNGs (192x192 and 512x512) for production and Play Store.
